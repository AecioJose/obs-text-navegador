Instruções de uso

Adicionar compra digite @fulano + o código assim: @fulano 9999

Adicionar 2, 3, ou mais basta colocar 2x, 3x antes do código assim: @fulano 2x 9999

Para adicionar um comentário, basta digitar /comentario que você quer após o códgio assim: @fulano 9999 /na cor azul

Para remover basta adicionar um "-" antes do código assim: @fulano - 9999

Para remover mais de 1, basta colocar -2x, -3x antes do código assim: @fulano -2x 9999

Para salvar o conteudo em PDF basta digitar /save

Para resetar e limpar tudo basta digitar /reset
